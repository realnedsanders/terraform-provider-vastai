# vastai Provider

The vastai provider enables Terraform to manage Vast.ai GPU compute resources, storage, networking, serverless endpoints, and account configuration.

## Authentication

The provider requires a Vast.ai API key for authentication. You can provide it via:

1. The `VASTAI_API_KEY` environment variable (recommended)
2. The `api_key` attribute in the provider configuration block

## Example Usage

```terraform
terraform {
  required_providers {
    vastai = {
      source = "realnedsanders/vastai"
    }
  }
}

provider "vastai" {
  # api_key can be set via VASTAI_API_KEY environment variable
  # api_key = "your-api-key-here"

  # api_url defaults to https://console.vast.ai
  # api_url = "https://console.vast.ai"
}
```
