# vastai_endpoints (Data Source)

Lists all serverless endpoints for the authenticated Vast.ai user. Returns endpoint details including autoscaling configuration and current state.

## Example Usage

```terraform
# List all serverless endpoints for the authenticated user
data "vastai_endpoints" "all" {}

output "endpoint_count" {
  value = length(data.vastai_endpoints.all.endpoints)
}
```
