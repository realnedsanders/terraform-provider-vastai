# vastai_templates (Data Source)

Search for Vast.ai templates by query string. Returns matching templates with their configuration details including Docker image, environment variables, and access settings.

## Example Usage

```terraform
# Search for templates matching a query
data "vastai_templates" "pytorch" {
  query = "pytorch"
}

output "template_count" {
  value = length(data.vastai_templates.pytorch.templates)
}
```
