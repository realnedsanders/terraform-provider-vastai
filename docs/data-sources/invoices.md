# vastai_invoices (Data Source)

Retrieves billing invoices for the authenticated Vast.ai user. Supports filtering by date range, type, and limit. Uses the v1 API endpoint.

## Example Usage

```terraform
# List billing invoices for the current month
data "vastai_invoices" "recent" {
  start_date = "2026-03-01"
  end_date   = "2026-03-31"
  limit      = 50
}

output "invoice_count" {
  value = length(data.vastai_invoices.recent.invoices)
}
```
