# vastai_cluster (Resource)

Manages a Vast.ai cluster. Clusters group machines into a private network with a designated manager node. Use `vastai_cluster_member` to add machines and `vastai_overlay` to create overlay networks on top of clusters.

## Example Usage

```terraform
# Create a cluster for private networking between machines
resource "vastai_cluster" "gpu_cluster" {
  subnet     = "10.0.0.0/24"
  manager_id = "12345"
}
```
