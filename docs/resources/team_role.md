# vastai_team_role (Resource)

Manages a Vast.ai team role with permission configuration. Roles define granular API permissions using a JSON permissions object. Roles are referenced by name when inviting team members.

## Example Usage

```terraform
# Define a team role with specific permissions
resource "vastai_team_role" "developer" {
  name = "developer"
  permissions = jsonencode({
    "api" = {
      "instance_read"  = {}
      "instance_write" = {}
      "template_read"  = {}
      "ssh_key_read"   = {}
    }
  })
}
```
