# vastai_api_key (Resource)

Manages a Vast.ai API key. API keys are immutable after creation -- changing name or permissions requires replacement. The key value is only available at creation time and cannot be recovered afterwards.

## Example Usage

```terraform
# Create a scoped API key with limited permissions
resource "vastai_api_key" "ci_deploy" {
  name        = "ci-deploy-key"
  permissions = jsonencode({
    "instances" : "full",
    "ssh_keys" : "read",
    "templates" : "read"
  })
}

output "api_key_id" {
  value = vastai_api_key.ci_deploy.id
}
```
