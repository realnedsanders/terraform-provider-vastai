# vastai_cluster_member (Resource)

Manages membership of a machine in a Vast.ai cluster. Uses a composite ID format of `cluster_id/machine_id`. Both the cluster ID and machine ID are immutable after creation.

## Example Usage

```terraform
# Create a cluster (required parent resource)
resource "vastai_cluster" "gpu_cluster" {
  name = "training-cluster"
}

# Add a machine to an existing cluster
resource "vastai_cluster_member" "worker_node" {
  cluster_id = vastai_cluster.gpu_cluster.id
  machine_id = "67890"
}
```
