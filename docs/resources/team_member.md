# vastai_team_member (Resource)

Manages a Vast.ai team member. Creating this resource sends an invitation to the specified email address. The member's user ID becomes the resource ID once they accept or when the invite is registered. Changing the email or role requires replacing the member.

## Example Usage

```terraform
# Invite a team member with a specific role
resource "vastai_team_member" "alice" {
  email = "alice@example.com"
  role  = vastai_team_role.developer.name
}
```
