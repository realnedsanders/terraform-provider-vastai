# vastai_network_volume (Resource)

Manages a Vast.ai network volume. Network volumes provide network-attached persistent storage that can be mounted across multiple instances. Network volumes are created from offers available via the `vastai_network_volume_offers` data source.

## Example Usage

```terraform
# Search for network volume offers
data "vastai_network_volume_offers" "nv_offers" {
  limit = 5
}

# Create a network-attached volume for shared storage across instances
resource "vastai_network_volume" "shared_data" {
  offer_id = data.vastai_network_volume_offers.nv_offers.most_affordable.id
  size     = 200
  name     = "shared-model-weights"
}
```
