# vastai_worker_group (Resource)

Manages a Vast.ai worker group. Worker groups bind to serverless endpoints and define the compute configuration including template, GPU requirements, and search parameters. Autoscaling behavior is controlled at the endpoint level via the `vastai_endpoint` resource.

## Example Usage

```terraform
# Create a serverless endpoint (required parent resource)
resource "vastai_endpoint" "llm_serving" {
  endpoint_name = "llm-inference"
}

# Create a worker group bound to the endpoint
resource "vastai_worker_group" "llm_workers" {
  endpoint_id   = vastai_endpoint.llm_serving.id
  template_hash = "abc123def456"

  search_params = "gpu_ram>=24 num_gpus=1 gpu_name=RTX_4090"
  gpu_ram       = 24.0
  test_workers  = 3
}
```
