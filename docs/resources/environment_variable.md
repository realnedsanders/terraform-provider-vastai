# vastai_environment_variable (Resource)

Manages a Vast.ai environment variable (secret). Environment variables are identified by their key name and the value is treated as sensitive. These are account-level secrets that can be referenced by instances.

## Example Usage

```terraform
# Store a secret as an account-level environment variable
resource "vastai_environment_variable" "hf_token" {
  key   = "HF_TOKEN"
  value = var.huggingface_token
}

variable "huggingface_token" {
  type      = string
  sensitive = true
}
```
