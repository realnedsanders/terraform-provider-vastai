    Create 2 networking resources for cluster management.

    **cluster/models.go:**
    - `ClusterResourceModel` struct: `ID types.String`, `Subnet types.String`, `ManagerID types.String`, `Timeouts timeouts.Value`

    **cluster/resource_cluster.go** (NETW-01):
    - `ClusterResource` struct with `client *client.VastAIClient`
    - Interface assertions: `resource.Resource`, `resource.ResourceWithImportState`
    - TypeName: `vastai_cluster`
    - Schema attributes:
      - `id`: StringAttribute, Computed, UseStateForUnknown
      - `subnet`: StringAttribute, Required, ForceNew. Description: "Subnet for the cluster (e.g., 10.0.0.0/24)."
      - `manager_id`: StringAttribute, Required, ForceNew. Description: "Machine ID to be the cluster manager."
      - `timeouts`: timeouts block
    - Create: Call client.Clusters.Create(subnet, managerID_as_int). The service returns a *Cluster from create-then-read (finds by subnet match). Store the ID.
    - Read: Call client.Clusters.List(), find by ID in the map. Update subnet from API. If not found, remove from state.
    - Update: Not supported (all fields ForceNew).
    - Delete: Call client.Clusters.Delete(id_as_int).
    - ImportState: Import by cluster ID.

    **clustermember/models.go:**
    - `ClusterMemberResourceModel` struct: `ID types.String` (composite: "cluster_id/machine_id"), `ClusterID types.String`, `MachineID types.String`, `NewManagerID types.String` (optional, for remove), `IsClusterManager types.Bool`, `LocalIP types.String`, `Timeouts timeouts.Value`

    **clustermember/resource_cluster_member.go** (NETW-03, per D-03: separate resource):
    - `ClusterMemberResource` struct with `client *client.VastAIClient`
    - Interface assertions: `resource.Resource`, `resource.ResourceWithImportState`
    - TypeName: `vastai_cluster_member`
    - Description: "Manages membership of a machine in a Vast.ai cluster. Uses composite ID format: cluster_id/machine_id."
    - Schema attributes:
      - `id`: StringAttribute, Computed. Description: "Composite ID: cluster_id/machine_id"
      - `cluster_id`: StringAttribute, Required, ForceNew
      - `machine_id`: StringAttribute, Required, ForceNew
      - `new_manager_id`: StringAttribute, Optional. Description: "If removing the cluster manager, specify the new manager machine ID."
      - `is_cluster_manager`: BoolAttribute, Computed
      - `local_ip`: StringAttribute, Computed
      - `timeouts`: timeouts block
    - Create: Parse cluster_id and machine_id to ints. Call client.Clusters.JoinMachine(clusterID, []int{machineID}). Set composite ID = "clusterID/machineID".
    - Read: Call client.Clusters.List(), find cluster by cluster_id, check if machine_id is in nodes list. If found, update is_cluster_manager and local_ip. If not found, remove from state.
    - Update: Not supported (ForceNew on key fields).
    - Delete: Parse new_manager_id if set. Call client.Clusters.RemoveMachine(clusterID, machineID, newManagerIDPtr).
    - ImportState: Import by "cluster_id/machine_id" composite string. Parse with strings.Split, set cluster_id and machine_id attributes.

    For both test files: httptest mock server, test Create, Read (via list), Delete. For cluster member: test composite ID parsing, test that nodes list is checked.
