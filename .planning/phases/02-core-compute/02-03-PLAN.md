    **Create internal/services/sshkey/resource_ssh_key.go:**
    Implement `resource.Resource` + `resource.ResourceWithImportState`:
    - `NewSSHKeyResource() resource.Resource` factory
    - `Metadata`: `resp.TypeName = req.ProviderTypeName + "_ssh_key"`
    - `Schema` per SCHM-01 through SCHM-06:
      - `id` StringAttribute Computed, `stringplanmodifier.UseStateForUnknown()` per SCHM-05
      - `ssh_key` StringAttribute Required, **Sensitive: true** per SCHM-02, validators:
        - `stringvalidator.LengthAtLeast(1)` per SCHM-01
        - `stringvalidator.RegexMatches(regexp.MustCompile(`^ssh-(rsa|ed25519|ecdsa|dsa) `), "must be a valid SSH public key starting with ssh-rsa, ssh-ed25519, ssh-ecdsa, or ssh-dsa")` per D-16
      - `created_at` StringAttribute Computed, `stringplanmodifier.UseStateForUnknown()` per SCHM-05
      - Timeouts block via `timeouts.Attributes(ctx)` with create=5min, read=5min, update=5min, delete=5min per SCHM-06
      - Every attribute has `Description` per SCHM-04:
        - id: "Unique numeric identifier of the SSH key."
        - ssh_key: "SSH public key content (e.g., 'ssh-rsa AAAA...'). Sensitive value."
        - created_at: "Timestamp when the SSH key was created."
    - `Configure`: Extract `*VastAIClient` from `req.ProviderData`, store in resource struct
    - `Create`:
      1. Read plan into SSHKeyResourceModel
      2. Call `client.SSHKeys.Create(ctx, model.SSHKey.ValueString())`
      3. Set ID from response (convert int to string via `strconv.Itoa`)
      4. Set CreatedAt from response
      5. Save state
    - `Read`:
      1. Get state into model
      2. Parse ID: `strconv.Atoi(model.ID.ValueString())`
      3. Call `client.SSHKeys.List(ctx)` (no single-get endpoint)
      4. Find key matching ID in the list
      5. If not found, remove from state (resp.State.RemoveResource)
      6. Update model fields from API response, save state
    - `Update`:
      1. Read plan into model
      2. Parse ID
      3. Call `client.SSHKeys.Update(ctx, id, model.SSHKey.ValueString())`
      4. Update model from response, save state
    - `Delete`:
      1. Read state into model
      2. Parse ID
      3. Call `client.SSHKeys.Delete(ctx, id)`
    - `ImportState`: `resource.ImportStatePassthroughID(ctx, path.Root("id"), req, resp)` per IMPT-01
